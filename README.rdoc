Learn How to Program

========================

This app was built from the lesson at: http://www.learnhowtoprogram.com/lessons/rebuild-this-site

To run this app: 
1. Clone the repo
2. Make sure you have Postgres installed and running
3. > bundle install
4. > rake db:create db:migrate db:test:prepare
5. > rails server
6. Navigate to localhost:3000
